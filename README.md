# mudra-campus-ledger
<br>
1.User Authentication: User login/signup karta hai (Manually verified).

2.Dashboard Entry: User ka personalized profile aur balance load hota hai.

3.Transaction Trigger: User "Transfer Money" ya "Fee Payment" action leta hai.

4.Data Validation: Backend check karta hai ki user ke paas sufficient balance hai ya nahi.

5.Ledger Update: Transaction successful hote hi database (SQLite) update hota hai.

6.Analytics Sync: Analytics engine transaction history ko read karke naya graph/chart generate karta hai.

7.Final Confirmation: User ko success/failure ka notification (Toast) dikhta hai.
